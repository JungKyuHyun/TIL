## 네트워크 정리

- 컴퓨터 네트워크: 컴퓨터와 컴퓨터가 통신 매체에 의해 연결되어서 데이터를 주고받는 것. 주고받는 정보는 비트로 표현된다
- 인터페이스: 2개의 서로 다른 시스템 사이에 존재하며 정보의 송수신을 중계하는 것
- 프로토콜: 데이터 통신상에서의 규칙
- 회선 교환: 교환기를 사용해 파이프를 교체해가며 통신하는 방식.회선이 점유되면 더 이상 접속되지 않아서 동시에 다수의 컴퓨터가 송수신할 수 없음. 주로 전화에서 사용됨
- 패킷 교환: 데이터를 일괄적으로 보내지 않고 여럿으로 분할해서 송수신하는 방식. 다수의 기기가 동시에 사용할 수 있음. 컴퓨터 네트워크는 패킷 교환 방식임
- 세그먼트(segment): 패킷 교환없이 케이블 분배기로 연결되는 범위
- 멀티엑세스 네트워크: 컴퓨터 한 대가 세그먼트 내의 모든 컴퓨터에 데이터를 송신할 수 있는 구조
- 포인트 투 포인트 네트워크: 컴퓨터 한 대가 다른 한 대의 컴퓨터에만 데이터를 송신할 수 있는 구조
- LAN(Local Area Network): 지역적으로 좁은 범위에서 본인이 책임지고 구축하는 네트워크
- WAN(Wide Area Network): 통신사업자의 서비스를 통해 떨어져 있는 지역의 LAN을 연결한 네트워크. 세계 최대 규모의 WAN은 인터넷임
